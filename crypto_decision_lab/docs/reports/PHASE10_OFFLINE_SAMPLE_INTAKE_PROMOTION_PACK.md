# QRDS/QOS Phase 10 Offline Sample Intake / Promotion Pack

Sprint 10I–10N is a bundled sample-intake and promotion-gate pack.

It covers:

- manual intake inbox;
- artifact sample generation from validated templates;
- file validation for JSONL/CSV;
- validated staging outputs under artifacts;
- canonical promotion blocked by default;
- station/status portal;
- documentation in the same package.

Primary launcher:

```bash
bash qrds_phase10_offline_sample_intake_promotion_pack_serve.sh
```

Generated portal:

```text
crypto_decision_lab/artifacts/phase10_offline_sample_intake_promotion_pack/index.html
```

This pack does not write into canonical data directories.
