# QRDS/QOS Phase 10 Depth Expansion Readiness Pack

Sprint 10U–10Z is a bundled depth-expansion readiness pack.

It covers:

- depth gap by symbol;
- manual/offline batch plan;
- source request files;
- inbox readiness;
- promotion blocked by default;
- station/status portal;
- documentation in the same package.

Primary launcher:

```bash
bash qrds_phase10_depth_expansion_readiness_pack_serve.sh
```

Generated portal:

```text
crypto_decision_lab/artifacts/phase10_depth_expansion_readiness_pack/index.html
```

This pack does not collect data and does not write into canonical data directories.
