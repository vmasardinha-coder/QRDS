# QRDS Phase 137 - Edge Candidate Eligibility Filter Research-Only

Gate: PHASE137_EDGE_CANDIDATE_ELIGIBILITY_FILTER_RESEARCH_ONLY_READY_RESEARCH_ONLY

Adds a research-only eligibility filter for unvalidated edge candidates.

Boundary:
- eligible only for research
- no edge validation
- no operational edge validation
- no trading eligibility
- no decision eligibility
- no signal
- no allocation
- no canonical write
