# QRDS Phase 171 - Shadow Readiness Requirement Registry Research-Only

Gate: PHASE171_SHADOW_READINESS_REQUIREMENT_REGISTRY_RESEARCH_ONLY_READY_RESEARCH_ONLY

Adds a research-only requirement registry for the shadow readiness synthesis layer.

Requirements:
- shadow score checkpoint required
- readiness synthesis required
- readiness explanation required
- readiness is not approval
- promotion remains blocked

Boundary:
- readiness remains descriptive-only
- no trading signal
- no recommendation
- no allocation
- no order payload
- no safe-apply
- no canonical write
