# QRDS/QOS — Data Audit Evidence Pack v1

Sprint 9D adds a research-only data audit evidence packet.

It records whether upstream reports and explicit dataset audit manifests provide enough evidence about row coverage, split coverage, missing values, duplicate bars, time-continuity gaps, lineage hashes, and upstream research context.

This layer cannot unlock operational use. It does not produce signals, orders, recommendations, allocation output, or live-fund workflows.

## Expected use

```bash
cd /workspaces/QRDS
bash qrds_data_audit_from_stack_serve.sh
```

The expected early result is incomplete, because explicit dataset audit manifests are not yet generated by the core research runner.
