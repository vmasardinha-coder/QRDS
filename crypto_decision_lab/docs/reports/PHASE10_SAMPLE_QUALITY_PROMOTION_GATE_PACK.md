# QRDS/QOS Phase 10 Sample Quality / Promotion Gate Pack

Sprint 10O–10T is a bundled sample-quality and promotion-gate pack.

It covers:

- staged sample quality metrics;
- per-symbol depth gap;
- duplicate and schema checks;
- artifact-stage quality gate;
- canonical promotion blocked by default;
- station/status portal;
- documentation in the same package.

Primary launcher:

```bash
bash qrds_phase10_sample_quality_promotion_gate_pack_serve.sh
```

Generated portal:

```text
crypto_decision_lab/artifacts/phase10_sample_quality_promotion_gate_pack/index.html
```

This pack does not write into canonical data directories.
