# QRDS Unified Dashboard Portal v1

Sprint 8K adds a single local portal that links the interpretation guide,
interactive dashboard and visual charts.

## Command from repository root

```bash
bash qrds_dashboard_portal.sh \
  --output-dir artifacts/dashboard_portal \
  --symbols BTC-USDT,ETH-USDT,SOL-USDT
```

Open via the printed server command.

## Output

```text
artifacts/dashboard_portal/index.html
artifacts/dashboard_portal/dashboard_serve_plan.json
```

## Recommended reading order

```text
1. guide
2. interactive dashboard
3. visual charts
```

## Safety

This is a local static research portal.

It does not produce allocation, signals, orders, recommendations, operational
decisions or real-capital actions.
