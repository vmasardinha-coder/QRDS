# Research Book Legacy Intake v1

Sprint 8Y aligns optional old book sources with the current 20-chapter QRDS/QOS research book plan.

## Usage

```bash
cd /workspaces/QRDS
bash qrds_research_book_legacy_intake_serve.sh \
  --output-dir artifacts/research_book_legacy_intake \
  --symbols BTC-USDT,ETH-USDT,SOL-USDT
```

Open with:

```text
Ports -> porta indicada -> Open in Browser / Open Preview
```

## Expected result without legacy files

`NO_LEGACY_BOOK_SOURCE_IMPORTED_YET_RESEARCH_ONLY`

This is not a failure. It means the intake lane is ready and waiting for old manuscript exports.
