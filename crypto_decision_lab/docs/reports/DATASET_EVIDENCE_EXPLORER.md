# QRDS Dataset Evidence Explorer v1

The Dataset Evidence Explorer is a research-only drilldown layer for local structured dataset evidence.

It answers:

- Which local structured dataset files are visible?
- Which symbols have local evidence?
- How many rows are visible?
- Are files readable?
- Do files expose time-column hints?
- Do files have content hashes for lineage?

It does not unlock trading, allocation, live execution, exchange access, or real-capital use.

Safety contract remains `INTERACTIVE_RESEARCH_ONLY`.
