# QRDS/QOS Phase 10 Offline Intake Validation Pack

Sprint 10D–10H is a bundled validation pack.

It covers:

- validation of the manual intake templates created in 10C;
- artifact-only staging manifest;
- no writes into canonical data directories;
- station status showing where the project is;
- documentation and portal in the same package.

Primary launcher:

```bash
bash qrds_phase10_offline_intake_validation_pack_serve.sh
```

Generated portal:

```text
crypto_decision_lab/artifacts/phase10_offline_intake_validation_pack/index.html
```

This pack does not ingest data into canonical directories.
